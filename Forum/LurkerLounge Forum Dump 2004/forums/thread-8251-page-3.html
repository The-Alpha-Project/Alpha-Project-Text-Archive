Hi,

Alright, we're gonna have Cleefy Cake Sunday at Noon Pac/3 PM Eastern.

I've noticed that the game response I get goes down steadily during the day. And it is not a function of how long I've been playing, because I log off and *reboot* about once every two hours or so. The slow down seems to be strongly dependent on the number of people on the server and how long it has been since the server was restarted. When Blizzard takes the server down (or when it crashes) the response speeds up again for a while. And none of this has anything to do with ping, which I almost always have below 200, even when the game grinds almost to a halt.

Putting all this together, I would suggest an earlier starting time and starting on Saturday instead of Sunday. The earlier starting time gives you a chance to get the instance done before response goes to hell. And unless the server crashes on Saturday night, it probably will not be restarted over the weekend.

I would also suggest that anyone having any side quests either ignore them or get them out of the way before the VC push. The longer it takes, the better the chances for the most powerful mob of all (lag) to kill the party.

Whatever y'all decide, I'll try (again) to make it. But after about 4 on weekdays and over most of the weekend, WoW turns into a slide show for me (on the "beta" server", I can usually play just fine on the PvP server -- but there I'm just a young Night Elf in a quite and secluded part of the world.)
