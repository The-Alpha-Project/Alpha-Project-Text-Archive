Quote:Now that we can't spot a lurker by tag, could you pin this thread so that it will be easy to check for fellow lurker characters?

I propose we keep a "Lurkers" chat room up. When you join the game with a character, type the following:

/join Lurkers

This will join the Lurkers chat room where everyone can talk to one another. If you want to talk in the chat room then type:

/1 <Text you want to send to the Lurkers chatroom>

(This assumes you've already turned off the general and trade channels. If you haven't then, you'd type "/3 <blah blah>" instead.

To see a list of the lurkers in the Lurker channel, type:

/chatlist 1
