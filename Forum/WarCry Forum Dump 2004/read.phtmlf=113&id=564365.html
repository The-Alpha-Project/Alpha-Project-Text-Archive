Cosmos, binding the World of Warcraft together
Hello Everyone!
Cosmos is a World of Warcraft UI Extension. It allows users to try an

abundant variety of features, such as:

Extra button bars (up to 60 buttons on the screen at one time!)

Increase Quest Text speed

Party Quests - share quest lists with your party.

Buff Timers

Self-casting

Timers & alerts

Macro tools

Combat statistics

Quest Minion tooltip

Totem Stomper - cast totems in sets!

In-game wanted ads

Map Notes - make marks on your world

Thottbot - get credit for being the first to find an item!
And a whole lot more! You can see our website at the Cosmos site

(http://www.thotthot.com). Documentation for Cosmos is under way at

(http://www.wowwiki.com/Cosmos). The Cosmos Beginner's guide is at

(http://www.wowwiki.com/Cosmos_Beginners_Guide)

Download
You can download Cosmos here:
http://down.thotthot.com/

The distributions:

 

Beta - should always work, but may not have the latest

bugfixes/features

Alpha - updated nightly, it may occasionally break or have new bugs.

Installation Instructions

Daily Zip:
Cosmos_Alpha.zip
This contains a nightly build of Cosmos, after being reviewed by our

closet of testers.

Take the Zip file and extract its Interface folder into World of

Warcraft directory. Then it should work!

We've automated all of our publishing tools, so that now, we don't have

to make new posts for you to enjoy everytime a small change has been

made.

However, if you want to know what version you're using type:
[/pre]
/cversion
[/pre]
Viola! the number appears.

Using the patcher
When you load the program, it will immediately update to patch in the

latest version that you've chosen! Simply go to Configure to choose

which version: Release, Beta, Alpha (Stable, Testing, Development) you

wish to use. Then viola, its patched.

As always, please enjoy our work, and feel free to leave new ideas here

to see! If you have problems, please try toggling off various options

in the Cosmos Configuration menu before posting!

Known issues:
If you forget to clean your directory out before running the EXE,

errors may occur. Be sure to save any custom mods that you have done

before installing Cosmos. If you forget, you may get loads of error

messages on start-up.

If your connection to the Cosmosui.org site is slow, then you will not

be able to get a distribution list. This is just the sad nature of the

world. Please try a ZIP mirror.

-Alex
