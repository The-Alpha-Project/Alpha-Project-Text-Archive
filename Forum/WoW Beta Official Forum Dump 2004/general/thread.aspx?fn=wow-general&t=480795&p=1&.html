 7. Re: accounts with horde and alliance? | 10/26/2004 6:51:54 PM GMTDT
The reason there are still many active accounts playing both sides is that those characters were made before the Blizzard policy was implemented. If you have both a Horde and an Alliance character on the PvP server (created BEFORE the policy was implemented) then you can play and create on both sides.

At retail, though, you will have to pick a side per server. On Server A, if you make a Horde character you will only be able to play Horde (unless you delete all characters) On Server B, you could play Alliance, but on that server you would only be able to play Alliance.

This policy (AFAIK) is only in effect for PvP servers. PvE has no such condition - you can play both sides on the same server.
