There's no reason for color coding to be complicated. It's best that it be as simple as possible.

Assuming the honor system remains, all you need to know from the color code is whether that player would be a dishonorable kill or not. So say WHITE for an opponent that would be an honorable kill, BLUE or GREY for one that would be dishonorable.
