As I've said elsewhere, I have been busy drafting up an explanation of the roles of various classes. This has dominated my time with the developers, as it is a touchy subject and an understanding of the philosophy behind each class is necessary. That said, I have not forgotten about these threads.

We did manage to get one response for you:

Currently, there is no big incentive for rogues to use daggers and use dagger specific attacks like backstab because there are better more powerful swords in the game that are more effective in PvP. Are there any plans to make rogues behave more according to their classic role instead of acting as dual wielding warriors?
This is a fine line to walk, and the developers will be looking further into this issue in the future.

There are some questions that simply cannot be answered at this time; others are more suggestions, not questions. If you have suggestions, I urge you to post them in other threads -- the designers do read these forums when seeking feedback or suggestions, and this thread is for questions, primarily.
