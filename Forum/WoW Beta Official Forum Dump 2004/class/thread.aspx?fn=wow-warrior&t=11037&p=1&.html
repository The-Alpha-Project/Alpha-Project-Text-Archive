 2. Re: Alternatives to taunt? | 10/19/2004 10:04:18 PM GMTDT
When you're in defensive stance, you can also use demoralizing shout (low rage cost), and challenging shout (once every 10 minutes without talents).

If you have tactical mastery, you can try switching to arms stance to use sweeping strikes and thunder clap, or berserker stance to use cleave and piercing howl (depending on talents).

If you have engineering, you could also try using bombs to get aggro.

Those are just some ideas for gaining/holding aggro.

If I have plenty of rage, I'll burn a demoralizing shout, and then thunderclap. If I'm low on agro I make sure to charge/demoralize. This is usually fine to get things started if everybody is targeting the same mob and not using any AoE spells.
