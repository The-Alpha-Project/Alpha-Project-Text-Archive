For soloing regular mobs (low armor/high armor):
Distract
Cold Blood + Ambush/Garrote
GS
Evis/Rupture
SS
Slice n Dice (to help proc cripple & deadly poison)
(...) SS/GS + anything else I want

For Instances:
Garrote
GS
(Fill to 5cp: + 1-2 SS's or if sealfate/initiative proced)
Kidney Punch (or Expose Armor if Sunder isn't on)
SS
Slice N Dice
(...) SS/GS + evis usually. Rinse + Repeat

For Soloing Elites (Balls out):
Distract
Coldblood + Ambush
GS

Rupture
Evasion
SS
Slice n Dice
(fill to 5cp: Thistle Tea + SS)
Preparation
Coldblood + evis
(...)
Keep slice n dice and evasion up when they wear off.
