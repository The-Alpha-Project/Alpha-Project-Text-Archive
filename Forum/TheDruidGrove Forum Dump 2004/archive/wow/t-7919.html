Having trouble logging into WoW? There's a new standalone patch available from the WoW community site (http://www.worldofwarcraft.com/).

Details on the patch:

We have become aware of a bug in our patching system that would have prevented future patches of the beta client from being delivered properly to all of our beta testers. In order to fix this, we need all beta testers to download a standalone patch. This patch will also allow us to test our standalone patching capabilities of the World of Warcraft beta client in case future updates need to be made via multiple methods, i.e. through the Blizzard Downloader, mirrored patch files, or by any alternate delivery systems.

<font color="red">Please make sure you have at least 3 GB of free space before you run this patch.</font>

For details on what balance changes have been made, please read the patch notes <a href="http://www.worldofwarcraft.com/patchnotes/patch-8-17-04.shtml">here</a>. To download the patch, <a href="http://www.worldofwarcraft.com/downloads/WoW-0.9.1.zip">PC users go here</a> and <a href="http://www.worldofwarcraft.com/downloads/mac.zip">Mac users go here</a>.

<b>If you have difficulty installing the patch, please unzip the patch file to the same directory in which you have World of Warcraft installed and run it from there.</b> PC users should have the patch file "WoW-0.9.1.exe." Mac users should have the file "WoW Mac 0.9.1 patch.dmg." If you experience difficulty installing the patch, please contact <a href="mailto:wowtech@blizzard.com">wowtech@blizzard.com</a>.
Edit: Updated info on patch problems!
We have identified a bug with the standalone beta patch that beta testers with fewer than 3 gigabytes of free space on their hard drives encountered when attempting to install the patch. Please see the <a href="http://www.worldofwarcraft.com/thread.aspx?ForumName=wow-tech-support&amp;ThreadID=17493">Technical Support</a> forum post here that describes the solution.
