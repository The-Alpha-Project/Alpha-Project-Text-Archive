Ok, we'll see how long these instructions last before they're removed, but here's a quick HOWTO for getting the .torrent file for the patch and using BitTornado to download it:


1) Get the trial of PE Explorer from http://www.heaventools.com/download.htm?r1=pex_nm


2) Install PE Explorer, run it and open up BlizzardDownloader.exe from your WoW directory.

3) Go to View -> Resources. Expand the TORRENT item, right click on one of the resources (I used 108) and go to Save Resource As. Save it to your desktop.

4) Rename the file you save as to something with a .torrent extension (I used blizz.torrent)

5) Load the .torrent file up in your Bittorrent client.

Now, I use BitTornado, and it had a problem with the tracker URL in the torrent file (I assume Blizzard has modified the format for their own purposes). The .torrent you get from the EXE has the following announce url:

http://12.129.233.80:3724/announce|http://12.129.233.81:3724/announce

Under Bittorrent's Advanced dialog I was able to use External Announce button to put in each of the announcers one at a time (http://12.129.233.80:3724/announce and then http://12.129.233.81:3724/announce)

I was getting about 20k up and down before, and I'm already getting 120k down and filling my upstream.

To Blizz: Sorry if you don't want people doing this, but I don't believe you've ever publicly stated that we MUST use your modified BT client. But now that you've removed the CD-Key requirement from your client, I see it as just being crippled. 3rd party BT clients are FAR better.

Good luck everyone!
