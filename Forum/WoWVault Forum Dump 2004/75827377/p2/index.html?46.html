Date Posted: 9/21 9:48am Subject: RE: I really hope Blizzard doesn't become Mythic as far as nerfing is concerned....
I don't care if Blizzard nerfs classes or items or anything - so long as they do it with the precision and care that they've had in their RTS games. Warcraft III stands out as a good example of *good* balance decisions - they're very slight, very subtle, and they serve a definite purpose. If the slight change is found to be insufficient, they change it some more - incrementally so until there is a consensus that balance has been achieved.

To see balancing done in a similar manner in WoW would be enough separation from Mythic for me. Rather than making entire classes obsolete with the addition of an item or 3 lines of text in the patch notes, I'm hoping any post-release(note: not beta) changes will be both slight and given lots of time to test on live servers prior to mucking with them again.
