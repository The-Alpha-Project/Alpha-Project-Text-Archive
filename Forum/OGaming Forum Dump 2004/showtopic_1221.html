mikititan

Group: Members

YES.

There is a very powerfull Librarian Admin Panel (only librarians see it) that allows us to check and review the book, and do the common functions very fast. There's also a message system with the librarians so users can receive information on what's going on, why the book can't be published, etc..

We demand a quality requirements for a book be published, so we only publish what is worth a check.

So, once a user has finished the book, he has to PUBLISH it so librarians can start reviewing it. After that process, the user is noticed wheter his book can be activated to appear in the public library or not.

We planned all this! :wink:
