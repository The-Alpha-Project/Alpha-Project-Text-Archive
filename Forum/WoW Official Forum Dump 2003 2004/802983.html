17. Re: Voice chat | 12/1/2003 1:39:09 AM GMTST
I'm starting a guild specifically because I want everyone in it to use voice chat. I've even gone so far as to get a Teamspeak server. I have always used voice chat in MMORPGs. In fact, I've used it for every single multiplayer game I've played since early '99. In MMORPGs, my interactions with others has been limited because it's hard to find others who use voice chat as well. Although my friends and I manage with people who don't use voice software, it's always so much easier when everyone in the group is using it...

That being said, it would be a nice feature (IMO) to have it built in.

Sure, there are plenty of external voice chat software packages, but having them external to the game greatly reduces the likelyhood that you will be able to convince people to use them. I mean what are the chances that the groups you meet up with are going to use voice chat. If they do, what are the chances that they will be using the same software as you? If they do use the same software, what are the chances that they will be willing to exit the game and take the time needed to load up the software and connect to your server.

Having the voice chat built right into the game does nothing but help (when it comes to actually using voice chat). I can't count the number of times people have told me that they'd love to use it except for the fact that they're only going to be on for a hour or so and don't want to bother getting everything setup.
