Damage Reduction

old formula, pre-beta5 AC scaling
(damage reduction) = (.3 * (armor class - 1)/(10*level+89))

Skill Points

450xp (base) = 1 skill point

XP Per Level

LevelXP = int((8*charlvl*mobxp)/100)*100

MobXP = (charlvl*5) + 45, for a mob of the same level as the character
