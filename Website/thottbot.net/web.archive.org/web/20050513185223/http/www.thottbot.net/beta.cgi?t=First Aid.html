Advantages of Bandages
 Reply Posted by Taedirk on 2004-10-24 20:50:25
Right now, bandages no longer channel when applying to yourself.  This means that if you can run out of combat far enough so the mob isn't hitting you, you can keep kiting it while a bandage heals you up (think Wing Clip / Hamstring + Bandage here).  Second, bandages work in combat while food requires you to be out of combat before you start eating.  Finally, you can apply bandages to others (which is still a channeled spell) as long as they aren't taking damage.
