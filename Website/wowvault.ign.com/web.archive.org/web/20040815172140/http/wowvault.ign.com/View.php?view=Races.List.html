Races

When creating a new character in World of Warcraft, the first choice a player must make is to pick a race. A character's race determines his or her appearance, faction, starting location, class choices, special abilities (for example, which mounts are available to the character), and starting stats.

The World of Warcraft is made up of two factions and eight races. The Humans, Night Elves, Dwarves, and Gnomes make up the Alliance, and the Orcs, Tauren, Undead, and Trolls make up the Horde. Below you can find links to more information about each race.
