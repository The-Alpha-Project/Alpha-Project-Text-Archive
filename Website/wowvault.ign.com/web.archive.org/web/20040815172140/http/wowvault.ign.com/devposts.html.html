Please note that these items are for users who have received the autentification key to be on the Fileplanet Stress Test ONLY. This does not apply to current ongoing beta test.

The following error:

'Sorry, the installer was unable to start up.

No installer data could be found'

can be caused by attempting to run the game installer from the .zip file. Please fully extract all of the files in the .zip you downloaded to a temporary location on the hard drive and then run the installer from there.

-----------------------

If you are encountering corrupt file errors when trying to unzip the stress test file downloaded from Fileplanet, please be sure that you are using the Windows XP handling of compressed folders, or the latest version of winzip:
http://www.winzip.com/
Many customers have reported issues when trying to use other decompression tools.
