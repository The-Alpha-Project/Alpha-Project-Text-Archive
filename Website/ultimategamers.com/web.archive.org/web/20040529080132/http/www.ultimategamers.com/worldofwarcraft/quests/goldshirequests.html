This page is intended to help you locate quests in the Goldshire region. Although we put every possible effort to make this list as complete as possible, we may have missed a few quests that are specific to a class or that have been added later on. As usual, we would appreciate your help making this page better and telling us what is missing, either a quest or an important information to complete it. Should you see something that needs to be either modified or added, please contact us. To see a list of all quests, please click on the link below.

Purple quests indicate incomplete data. Help appreciated.
