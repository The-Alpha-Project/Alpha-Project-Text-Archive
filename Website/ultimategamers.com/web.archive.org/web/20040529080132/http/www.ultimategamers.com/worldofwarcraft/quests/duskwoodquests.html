This page is intended to help you locate quests in the Duskwood area. Although we put every possible effort to make this list as complete as possible, we may have missed a few quests that are specific to a class or that have been added later on. As usual, we would appreciate your help making this page better and telling us what is missing, either a quest or an important information to complete it. Should you see something that needs to be either modified or added, please contact us. A list of all quests is available using the link below. As a final note, the levels indicated are those at which we believe you can receive the quest, not necessarily those at which you can complete them.
