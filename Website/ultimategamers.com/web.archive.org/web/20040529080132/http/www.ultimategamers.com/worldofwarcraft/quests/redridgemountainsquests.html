This page is intended to help you locate quests in the Redridge Mountains region. Although we put every possible effort to make this list as complete as possible, we may have missed a few quests that are specific to a class or that have been added later on. As usual, we would appreciate your help making this page better and telling us what is missing, either a quest or an important information to complete it. Should you see something that needs to be either modified or added, please contact us. A list of all quests is available using the link below.

The level indicated for these quests is far from accurate, as several characters were used to assess it.
